## Introduction

System module with CaseIterable.

- Foundation
  - Locale
- UIKit
  - UIFont

## Example

To run the example project, clone the repo, and run `xcodegen && pod install && open CaseIterable.xcworkspace` from the Example directory first.

## Requirements

- CocoaPods >= 1.7.2
- Swift >= 5.0
- Xcode >= 10.2
- XcodeGen
